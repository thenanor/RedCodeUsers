Welcome to the Users App

In this app you will be able to do CRUD operations around users.
You will be able to create a new user
View the list of existing users
Update and delete each one of them by clicking on the View button from the list

 

Technical Details

  • Back-end Details
  • Front-end Details
  • Not Implemented
  • The Back-end is built with: ASP.NET Core 2.0
  • Entity Framework Code first Migration is used as the ORM
  • Used: API Resources (DTOs) and Auto Mapper
  • Applied:
    • Repository Pattern and Unit Of Work
    • Server-Side Table Filtering / Sorting / Paging
  • The Front-end is built with: Angular 4.2.5
  • Additional Installed Packages:
    • Toast Notification
    • Bootstrap and Font Awesome
    • Pagination
    • Underscore
  • Authentication and Authorization
  • Custom Validations for user registration form
